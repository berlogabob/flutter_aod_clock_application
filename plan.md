# Nightstand AOD Clock — Project Plan

## Status: v0.0.1 — MVP complete

## Current Features (Done)
- [x] Full-screen time display (HH:mm) with OpenDyslexicM Nerd Font
- [x] Wakelock to keep screen on
- [x] Pure black AMOLED background
- [x] Immersive fullscreen (no status/nav bars)
- [x] Portrait orientation lock
- [x] Efficient timer (synced to minute boundary, fires once/min)
- [x] AMOLED burn-in prevention (pixel jitter every 5 min)
- [x] Error-safe wakelock handling

## Backlog / Future Ideas
- [ ] Screen brightness control (dim to ~10% at night)
- [ ] Battery-aware wakelock (only keep screen on while charging)
- [ ] Optional date display (below the time, smaller font)
- [ ] Tap-to-toggle seconds (show for 5 s, then hide)
- [ ] Settings screen (font size, font family, color themes)
- [ ] Widget tests
